# Roadmap

## v0.1

- Rule model for AI-assisted software estimates.
- Dependency-free CLI.
- Codex skill adapter.
- Example estimates.
- Unit tests and GitHub Actions.

## v0.2

- Claude Code adapter.
- Cursor rules adapter.
- Windsurf rules adapter.
- Custom CLI rule file support.
- More real-world estimate examples.
- Better multi-component estimates.

## v0.3

- Parse task descriptions into multiple estimate components.
- Export estimate reports as Markdown.

## v0.4

- Community calibration dataset.
- Compare estimated vs actual outcomes.
- Derive default compression ranges from evidence.

## Later

- Linear/Jira/GitHub Issue integrations.
- Web interface for non-CLI users.
- Organization-specific calibration profiles.
