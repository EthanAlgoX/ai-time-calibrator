# Example: CRUD API

## Request

Add CRUD endpoints for a `Project` resource in an existing backend with similar resources already implemented.

## Traditional Estimate

3 days

## AI-Adjusted Estimate

- Optimistic: 0.75 day
- Expected: 1.25 days
- Conservative: 2 days

## Why It Changed

- Existing patterns make route, model, validation, and test generation highly AI-compressible.
- AI can draft repetitive endpoint code and tests quickly.
- Review, integration, and regression checks still need time.

## Risks

- Authorization rules may differ from existing resources.
- Database migration review may add coordination time.
- Missing acceptance criteria could widen the estimate.
