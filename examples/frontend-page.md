# Example: Frontend Page

## Request

Build a settings page with editable profile fields, notification toggles, validation states, and responsive layout.

## Traditional Estimate

4 days

## AI-Adjusted Estimate

- Optimistic: 1.5 days
- Expected: 2 days
- Conservative: 3 days

## Why It Changed

- Component scaffolding, form state, validation, and tests are AI-compressible.
- Existing design system patterns can be reused faster with AI assistance.
- Visual QA, accessibility checks, and product polish remain meaningful costs.

## Risks

- Design ambiguity can increase iteration time.
- Edge cases around validation and saved state may require manual testing.
- If no design system exists, conservative estimate should be used.
