# AI Time Calibrator v0.1.0

First public release of AI Time Calibrator, an open framework for calibrating software development estimates for AI-assisted workflows.

## Highlights

- Defines a practical AI-assisted estimation model.
- Adds default task type calibration rules.
- Adds risk factor and output model references.
- Provides a dependency-free CLI.
- Supports text and JSON output.
- Supports custom task type rules with `--rules`.
- Includes Codex, Claude Code, Cursor, and Windsurf adapters.
- Includes example estimates.
- Adds tests and GitHub Actions CI.
- Adds contribution guide, roadmap, changelog, issue templates, and PR template.

## Quick Start

```bash
python3 scripts/estimate.py --list-task-types
python3 scripts/estimate.py --traditional-hours 24 --task-type crud_api
python3 scripts/estimate.py --traditional-hours 24 --task-type crud_api --format json
```

## Status

This is an early v0.1 release. The default calibration ranges are heuristic and should be improved with real-world examples over time.
