# AI Time Calibrator v0.1.1

Small usability release after v0.1.0.

## Highlights

- Adds Markdown CLI output for copying estimates into issues, PRs, and planning tools.
- Adds Chinese README.
- Adds an example calibration dataset structure.
- Keeps the CLI dependency-free.

## Quick Start

```bash
python3 scripts/estimate.py --traditional-hours 24 --task-type crud_api --format markdown
```

## Validation

```bash
python3 -m py_compile scripts/estimate.py
python3 -m unittest discover -s tests
```
