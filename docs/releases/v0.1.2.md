# AI Time Calibrator v0.1.2

Small stability and tooling release.

## Highlights

- Adds `--output` for writing CLI output directly to a file.
- Adds JSON Schema files for task type rules, estimate output, and dataset records.
- Adds schema file tests.
- Updates README and Chinese README with file output usage.

## Quick Start

```bash
python3 scripts/estimate.py \
  --traditional-hours 24 \
  --task-type crud_api \
  --format markdown \
  --output report.md
```

## Validation

```bash
python3 -m py_compile scripts/estimate.py
python3 -m unittest discover -s tests
```
